# Brawl Roulette

Фанатская рулетка бойцов Brawl Stars. Пользователь выбирает фильтры, нажимает кнопку и получает случайного бойца.

## Что в проекте

- `index.html` — вся страница в одном файле: разметка, стили, база бойцов, логика рулетки и звуки.
- `vercel.json` — настройки для корректной отдачи статического сайта на Vercel.
- `.gitignore` — исключает лишние служебные файлы.

## Что исправлено в этой версии

- На мобильных кнопка запуска перенесена прямо в блок рулетки.
- Рулетка, кнопка запуска, кнопка фильтров и карточка результата помещаются в один компактный блок.
- Большой текст и статистика на мобильных скрываются, чтобы первый экран был удобным.
- Для маленьких экранов рулетка автоматически становится ниже.
- Технических комментариев на странице нет.

## Проверка на компьютере

1. Распакуй архив проекта.
2. Открой папку `brawl-roulette-vercel`.
3. Дважды нажми на `index.html`.
4. Проверь страницу в браузере.
5. Включи мобильный режим в DevTools и проверь ширину 375–430 px.

Также можно запустить локальный сервер:

```bash
cd brawl-roulette-vercel
python3 -m http.server 3000
```

Потом открыть:

```text
http://localhost:3000
```

## Загрузка на Vercel через GitHub

1. Создай аккаунт на GitHub, если его ещё нет.
2. Нажми `New repository`.
3. Назови репозиторий, например `brawl-roulette`.
4. Загрузи в репозиторий файлы из папки `brawl-roulette-vercel`:
   - `index.html`
   - `vercel.json`
   - `.gitignore`
   - `README.md`
5. Открой Vercel.
6. Нажми `Add New` → `Project`.
7. Подключи GitHub.
8. Выбери репозиторий `brawl-roulette`.
9. В настройках проекта оставь:
   - Framework Preset: `Other` или `No Framework`
   - Build Command: пусто
   - Output Directory: пусто
10. Нажми `Deploy`.
11. После деплоя Vercel выдаст ссылку на сайт.

## Загрузка через Vercel CLI

Установи Vercel CLI:

```bash
npm i -g vercel
```

Авторизуйся:

```bash
vercel login
```

Перейди в папку проекта:

```bash
cd brawl-roulette-vercel
```

Загрузи сайт:

```bash
vercel --prod
```

После завершения команда покажет ссылку на опубликованную страницу.

## Как быстро заменить текст

Открой `index.html` и найди нужную фразу через поиск по файлу.

## Как обновить список бойцов

Открой `index.html`, найди массив `BRAWLERS` и измени данные внутри него.
