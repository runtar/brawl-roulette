# Brawl Roulette

Готовый статический сайт для Vercel.

## Что внутри

- `index.html` — разметка страницы
- `styles.css` — адаптивный дизайн
- `app.js` — логика рулетки, фильтров, звуков и конфетти
- `brawlers.json` — база бойцов для удобного обновления
- `vercel.json` — базовая конфигурация Vercel

## Как запустить локально

Откройте `index.html` в браузере или запустите простой локальный сервер:

```bash
python -m http.server 3000
```

Затем откройте `http://localhost:3000`.

## Как загрузить на Vercel

1. Создайте аккаунт или войдите на vercel.com.
2. Создайте новый проект через Add New → Project.
3. Импортируйте репозиторий GitHub с этими файлами.
4. В настройках проекта выберите Other или оставьте автоопределение.
5. Build Command оставьте пустым.
6. Output Directory оставьте пустым или укажите `.`.
7. Нажмите Deploy.

После деплоя Vercel даст ссылку вида `https://project-name.vercel.app`.
